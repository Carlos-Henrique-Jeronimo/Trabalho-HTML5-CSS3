  • O que é um HD?

    • Popularmente chamado de HD (Hard Disk Drive), é o componente do computador onde são gravados e armazenados os dados permanentemente. Trata-se de uma memória não-volátil, pois quando se desliga o computador, as informações não são perdidas, tal como acontece com as memórias RAM.
